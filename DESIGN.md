# Design principles (standing instruction)

Apple-grade design system and quality throughout.
- System font stack, clear typographic hierarchy, generous whitespace (8pt scale).
- Restrained neutral palette + one accent; strong text contrast (WCAG AA).
- Large radii, soft layered shadows, subtle motion/transitions.
- Image-forward, consistent cards; polished modal with backdrop blur.
- No broken links anywhere; only show a country link if it resolves to that exact product.
- Accessibility: keyboard, focus states, alt text, aria labels.
